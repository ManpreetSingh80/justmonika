Just Monika Screen Saver

Download the chrome extension and make monika as your screen saver



Features

  • Wait time after machine becomes idle to display screensaver
  • Background sound
  • A scheduler to control when the screensaver, display, and computer should remain on
  • Do not display over full screen Chrome windows (e.g., YouTube, Netflix)
  • Show on all displays
  • Display time
Works great on all platforms.

How to use

  • Add extension to your browser from above links
  • Left click on the extension icon
  • Configure the settings
    Wait time - Screensaver will appear when browser is left idle(no keyboard event, no mouse event) for this much time
  • Now left the browser idle for some time. Don't touch your keyboard and mouse.
    Monika will appear after wait time( For ex. after 5 minutes as set in above image)
    Note - Screen saver will not appear if browser is in full screen mode. This can be disabled in Display settings
